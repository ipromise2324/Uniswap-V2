# Uniswap-V2-SwapTokens

Let's learn about DeFi. This repo explains how to trade an ERC20 token for another using a decentralized exchange called Uniswap.

## Settings

1. Install dependency

   ```bash
   yarn install
   ```

2. Fill up .env

   ```bash
   cp .env.example .env
   ```

3. Run

   ```bash
   yarn hardhat test
   ```
